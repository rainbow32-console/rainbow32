Keybinds

Global

Ctrl+S: Save project

Ctrl+O: Open project

Alt+N: Create a new project

Ctrl+Shift+→: Move to the right tab

Ctrl+Shift+←: Move to the left tab

Doubleclick on any window: Minimize or maximize the window

Drawing Tool

←/↑/→/↓: Palette selection move

Alt++/Alt+-: Zoom in/out

Masking tool

Alt++/Alt+-: Zoom in/out

Music Tool

←/→: Selection left/right

↑/↓: Octave up/down

Alt+↑/Alt+↓: Selected channel up/down

+/-: Increase/decrease the sounds length

Space: Play/pause

Escape: Clear current note

Key on the bottom of the keyboard: Set current note to key
